Les bases de SASS & SCSS

  • Installer SASS via npm "npm install _g sass"
  • Pour vérifier qu'il estbien installé, lancer "sass" ou "npx sass"
  • Si l'éxécution des scripts est bloqué, lancer dans le powershell "set-executionpolicy unrestricte"
  • Les différents commandes
  • sass pathToFile.scss pathToFile.css
  • sass --watch pathToFile.scss pathToFile.css
  • sass --watch sassFolder:cssFolder
